Hotwire Native unlocks mobile apps for web developers. Build your HTML views once and deploy them to mobile web, Android, and iOS simultaneously.

It works by wrapping an embedded web view with native animations. HTML is fetched from your server like a traditional browser and rendered in the hybrid app.

Reusing your existing mobile web content is often good enough for a variety of screens and interactions. And when you need more interaction or fidelity, Hotwire Native has you covered.

  • Strada provides the glue to render web components as platform-specific native ones. Like a web dialog rendering as an Action Sheet on iOS or a BottomSheetDialog on Android.
  • Path Configuration maps endpoints to behavior, providing hooks to render fully native screens with full access to underlying iOS and Android SDKs and APIs.
